COMMAND REFERENCE
=================

1. CREATE REPOSITORY
   npm run cli -- create-repository --name "AliceModel" --price 0
   
   Output: Repository ID and Capability ID (save these!)

2. COMMIT MODEL
   npm run cli -- commit \
     --repo <REPO_ID> \
     --cap <CAP_ID> \
     --branch main \
     --message "Initial commit" \
     --file model.bin \
     --accuracy 98.5 \
     --loss 0.02 \
     --epochs 100 \
     --f1-score 0.92

3. SHOW CERTIFICATE
   npm run cli -- certificate --repo <REPO_ID>
   
   Output: Cryptographic provenance certificate with trust score

4. VERIFY MODEL
   npm run cli -- verify --repo <REPO_ID>
   
   Output: Dramatic "VERIFIED" banner with trust level

5. FORK REPOSITORY
   npm run cli -- fork --from <ALICE_REPO_ID> --name "BobsModel" --price 1000000000
   
   Output: New repo with upstream_author locked to Alice

6. VIEW MARKETPLACE
   npm run cli -- storefront
   
   Output: Table of all repositories with trust scores

TROUBLESHOOTING
===============

Build errors:
  npm install
  npm run build

Command not found:
  Make sure you're using: npm run cli -- <command>
  Or: node dist/index.js <command>

Transaction fails:
  - Check .env has SUI_PRIVATE_KEY
  - Check .env has PACKAGE_ID
  - Ensure you have SUI tokens for gas

Verification fails:
  - Make sure you committed a version first
  - Check repository has a main branch
  - Verify file hasn't been modified
