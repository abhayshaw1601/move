TROUBLESHOOTING
===============

If commands fail:
1. Check .env file has correct SUI_PRIVATE_KEY and PACKAGE_ID
2. Ensure you have SUI tokens for gas
3. Verify repository IDs are correct
4. Run: npm run build

If verification fails:
1. Make sure you committed a version first
2. Check that the repository has a main branch
3. Verify the file hasn't been modified

For demo practice:
1. Run through the script 2-3 times
2. Have repository IDs ready in a text file
3. Test both terminals work independently
4. Time yourself to stay under 3 minutes


TIPS FOR SUCCESS
================

1. Keep terminal font size large (18-20pt) for screen recording
2. Use high contrast color scheme
3. Clear terminals before starting: cls (Windows) or clear (Linux/Mac)
4. Have all repository IDs copied and ready to paste
5. Practice the narration timing
6. Keep mouse movements minimal and deliberate
7. Pause briefly after each dramatic reveal (trust score, access denied)
8. End with the GitHub link visible on screen
